# xdcipher

A Rust application that converts images and videos to ASCII art in real-time. Built with [egui](https://github.com/emilk/egui) for the native desktop GUI and compiles to WebAssembly for web deployment via Cloudflare Pages.

## Features

- **Image-to-ASCII** — Convert images (PNG, JPEG, GIF, WebP, BMP, TIFF) into ASCII art with adjustable width up to 5000 characters.
- **Video-to-ASCII** — Decode video files via ffmpeg and play the ASCII result in real-time with playback controls (native only).
- **WASM Video Support** — Video-to-ASCII via browser `<video>` element capture on the web build.
- **Real-time Configuration** — Sidebar sliders for width, height, contrast, edge threshold, and font size, all updating instantly.
- **Edge Detection** — Sobel-based edge detection that renders directional edge characters (`-`, `/`, `|`, `\`).
- **Floyd-Steinberg Dithering** — Improves perceived detail with error diffusion dithering.
- **Color Output** — Toggle between monochrome and color ASCII preserving original pixel colors.
- **Invert Mode** — Swap black/white backgrounds.
- **Multiple Charsets** — Choose from 10-step, Compact, Bourke 70, or Unicode Blocks.
- **Zoom & Pan** — Zoom in/out of the ASCII board and scroll freely in both axes.
- **Collapsible Sidebar** — Toggle the control panel for a full-screen view of the output.
- **Save Output** — Export ASCII art as `.txt` or styled `.html`.
- **Drag & Drop** — Drop image files directly onto the window.
- **Cross-platform** — Native desktop (macOS, Windows, Linux) and WASM web build.

## Running Natively

```sh
cargo run --bin xdcipher
```

Video playback requires `ffmpeg` and `ffprobe` to be installed and available on `PATH`.

## Building for Web

```sh
./build-web.sh
```

This compiles the project to WASM with `wasm-bindgen` and outputs the bundle to `web/pkg/`. To serve locally:

```sh
python3 -m http.server 8080 -d web
```

Then open `http://localhost:8080` in a browser.

## Deploying to Cloudflare Pages

1. Push or upload the `web/` directory (including `web/pkg/`) to a Cloudflare Pages project.
2. Set the build output directory to `web`.
3. No additional build step is needed if you commit the pre-built WASM artifacts.

Alternatively, use `wrangler`:

```sh
npx wrangler pages deploy web --project-name <project>
```

## Project Structure

```
src/
  lib.rs        — Library root, WASM entry point (WebHandle)
  app.rs        — egui UI, file handling, video playback loop
  engine.rs     — ASCII conversion, Sobel edges, dithering, HTML export
  video.rs      — ffmpeg-based frame decoder (native only)
web/
  index.html    — WASM bootstrap page
  pkg/          — Compiled WASM output (generated by build-web.sh)
```
